# nextToDo

> Web Todo应用

## Tech Stack

- HTML/CSS/JavaScript
- Vue
- Webpack
- Node.js/Express
- Docker/Nginx
- PM2

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload
npm run dev

# build for production with minification
npm run build
```
